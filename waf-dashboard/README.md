# 📑 YMCS Shield Dashboard

This is the React-based frontend dashboard for the **YMCS Shield WAF** project. It provides real-time visualization of security incidents, policy management, and interaction with the AI Security Analyst.

## 🚀 Features

- **Real-Time Visualization**: Powered by Recharts for incident metrics and traffic analysis.
- **AI Analyst Interface**: Interactive chat to query Gemini-powered security forensics.
- **Security Policy Center**: Dynamic toggles for blocking SQLi, XSS, and managing rate limits.
- **Custom Reporting**: Request and download PDF security reports.

## 🛠️ Tech Stack

- **React.js**: Core framework.
- **Framer Motion**: Smooth UI transitions.
- **Lucide React**: Vector icons.
- **Axios**: API communication with the backend.

## 🚦 Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
   Open `http://localhost:3000` to view the dashboard.
